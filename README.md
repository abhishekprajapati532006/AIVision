# AIVision

AIVision is a simple imagen recognition demo.  
It using python and tensorflow but not full working. This project is for learnning.

## Feautures
- Detects objects from images
- Saves resutls to local file
- Has sample images but not all working

## Requirements
- Python3
- tensoflow
- pillow

## Instalation
1. Clone the repo
2. run `pip install -r requirements.txt`
3. then run `python main.py`

## Usage
Just put your image in /images folder and run the script.  
Output will be saved in results.txt
